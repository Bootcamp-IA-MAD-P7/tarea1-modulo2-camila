## 1️ - ¿Qué es el EDA y cuál es su propósito en el análisis de datos?

El **Análisis Exploratorio de Datos (EDA)** es un proceso utilizado para examinar, comprender y resumir las principales características de un conjunto de datos antes de realizar análisis más avanzados o construir modelos de Machine Learning.

### 🎯 Objetivos principales

* Identificar patrones.
* Detectar anomalías o valores atípicos.
* Encontrar relaciones entre variables.
* Evaluar la calidad de los datos.
* Preparar la información para etapas posteriores.

---

## 2️ - ¿Qué tipos de datos existen en un EDA?

En un EDA se trabaja principalmente con dos grandes categorías de datos:

### 🔢 Datos numéricos

* Discretos
* Continuos

### 🏷️ Datos categóricos

* Nominales
* Ordinales

La correcta identificación del tipo de dato permite seleccionar las técnicas de análisis y visualización más adecuadas.

## 3️ - ¿Cuál es la diferencia entre análisis univariado, bivariado y multivariado?

| Tipo de análisis | Descripción                                  |
| ---------------- | -------------------------------------------- |
| 📌 Univariado    | Analiza una única variable                   |
| 📌 Bivariado     | Analiza la relación entre dos variables      |
| 📌 Multivariado  | Analiza tres o más variables simultáneamente |

### Ejemplos

* **Ventas mensuales** → Univariado.
* **Ventas vs Publicidad** → Bivariado.
* **Ventas + Publicidad + Precio + Clientes** → Multivariado.

## 4️ - ¿Qué es la estadística descriptiva?

La estadística descriptiva permite recopilar, organizar, resumir y describir un conjunto de datos mediante medidas numéricas y representaciones gráficas.

### 📈 Medidas principales

#### Tendencia central

* Media
* Mediana
* Moda

#### Dispersión

* Rango
* Varianza
* Desviación estándar

#### Visualización

* Histogramas
* Diagramas de caja
* Gráficos de barras

## 5️ - ¿Qué es la limpieza de datos?

La limpieza de datos consiste en preparar los datos para su análisis o modelado corrigiendo errores e inconsistencias.

### Tareas habituales

* Manejo de valores nulos.
* Eliminación de duplicados.
* Detección de outliers.
* Corrección de formatos.
* Normalización y estandarización.

## 7️ - ¿Qué es una matriz de correlación?

Una matriz de correlación muestra la relación existente entre distintas variables de un conjunto de datos.

### 📌 Interpretación

| Valor | Significado                   |
| ----- | ----------------------------- |
| 1     | Correlación positiva perfecta |
| 0     | Sin correlación lineal        |
| -1    | Correlación negativa perfecta |

### Utilidad

* Detectar dependencias.
* Seleccionar variables relevantes.
* Comprender relaciones entre características.

---

## 8️ - ¿Qué son los outliers?

Los outliers o valores atípicos son observaciones que se alejan significativamente del resto de los datos.

### 🔍 Métodos de detección

* Boxplot
* Histograma
* Z-Score
* IQR

### 🛠️ Tratamiento

* Eliminación
* Imputación
* Transformaciones
* Modelos robustos

---

## 9️ - ¿Qué es Hypothesis Testing?

El **Hypothesis Testing** o prueba de hipótesis es un método estadístico que permite tomar decisiones sobre una población utilizando datos de una muestra.

### 📌 Conceptos principales

* **H₀ (Hipótesis nula)**
* **H₁ (Hipótesis alternativa)**

### Aplicación en EDA

* Comparar grupos.
* Validar supuestos.
* Analizar relaciones entre variables.
* Complementar el análisis exploratorio con evidencia estadística.

## 📚 Conclusión

El Análisis Exploratorio de Datos constituye una etapa fundamental dentro de cualquier proyecto de ciencia de datos, ya que permite comprender la estructura de la información, detectar problemas de calidad y obtener conocimientos valiosos antes de aplicar modelos predictivos o técnicas avanzadas de análisis.
